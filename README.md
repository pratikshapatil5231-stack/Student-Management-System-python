# 🎓 Student Management System (Python)

## 📖 Description
Student Management System is a simple Python console application that helps manage student records. It allows users to add, view, search, and delete student information. The program also calculates total marks, average, and grade automatically.

## ✨ Features
- ✅ Add Student
- ✅ View Student Details
- ✅ Search Student
- ✅ Delete Student
- ✅ Calculate Total Marks
- ✅ Calculate Average Marks
- ✅ Generate Grade
- ✅ Menu-Driven Program
- ✅ Uses Functions and Modules
- ✅ User-Friendly Console Interface

## 🛠️ Technologies Used
- 🐍 Python 3
- 💻 VS Code

## 📂 Project Structure
- 📄 main.py
- 📄 menu.py
- 📄 add_student.py
- 📄 calculate_marks.py
- 📄 grade.py
- 📄 display_student.py
- 📄 search_student.py
- 📄 delete_student.py

## 📸 Output
- ➕ Add student details
- 👀 View all students
- 🔍 Search student by name
- 🗑️ Delete student by name
- 📊 Display total, average, and grade
